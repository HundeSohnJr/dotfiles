# 🎨 My Dotfiles

Personal dotfiles for a clean, productive development environment.

## Features

- 🚀 Starship prompt (fast & beautiful)
- 🐚 Zsh configuration with useful aliases
- 🎨 Git-aware prompt with branch info
- 📦 Auto-installs in GitHub Codespaces

## Manual Installation
```bash
git clone https://github.com/YOUR-USERNAME/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Included

- `.zshrc` - Zsh configuration
- `starship.toml` - Starship prompt config
- `install.sh` - Auto-setup script
